# Dataset

The Scene-Level Grounding Benchmark dataset is released separately on HuggingFace:

**URL**: `<DATASET_URL>`

The dataset contains:
- 26 goal-pattern markdown specifications (`patterns/`)
- Ground-truth Unity scene representations in 6 formats (`ground_truth/`)
- V2 (behavior-only) and V4 (full-scene) IR files (`ir/`)
- 858 LLM generation outputs across 4 models × 3 conditions × 3 seeds (`generation_outputs/`)
- Prerecorded Unity replay logs for all 858 attempts (`replay_logs/`)
- Pre-computed evaluation metrics (`metrics/`)
